# Cat vs Dog Classifier

Бинарный классификатор изображений на основе свёрточной нейронной сети (CNN), обученной на датасете Cats vs Dogs. Развёрнут как веб-приложение на Streamlit.

## 📊 Результаты модели

| Метрика | Значение |
|---|---|
| Train Accuracy | ~82% |
| Val Accuracy | ~82% |
| Архитектура | Custom CNN |
| Размер модели | 20 MB |


## Структура проекта

```
cat-dog-classifier/
├── app.py                      # Streamlit веб-приложение
├── cnn.py                      # Архитектура модели

├── render.yaml                 # Конфиг деплоя Render
└── README.md
```
## Архитектура

```
Input (3×128×128)
  → Conv2d(3→32, 3×3) → ReLU → MaxPool(2×2)
  → Conv2d(32→32, 3×3) → ReLU → MaxPool(2×2)
  → Flatten → Linear(28800→128) → ReLU → Dropout(0.5)
  → Linear(128→2)
  → Softmax
```

# Вывод по проекту: Cat vs Dog Classifier

## Что было сделано

Разработан бинарный классификатор изображений на основе свёрточной нейронной сети (CNN) с нуля. Модель обучена на датасете Cats vs Dogs и развёрнута как веб-приложение на Streamlit + Render.

## Результаты

| Метрика | Значение |
|---|---|
| Train Accuracy | ~82.4% |
| Val Accuracy | ~82% |
| Val Loss | ~0.4105 |
| Gap| 0.0041|
| Эпох обучения | 25 |

## Архитектурные решения

В ходе работы были протестированы несколько вариантов архитектуры. Оптимальным оказался простой вариант: два свёрточных слоя (Conv2d 3→32→32), MaxPooling, один полносвязный слой (28800→128) и Dropout(0.5).

## Итоги и выводы

- Построенна высокопроизводительная и точная модель в рамках используемой методологии.
- Агрессивная аугментация ухудшает обучение.
- Умеренная аугментация даёт стабильный результат.
- Улучшены показатели базовой модели и модели испольщованной в tensorflow.
- Потолок кастомной CNN на данном датасете — ~82–83% val accuracy

## Дальнейшее развитие

Для преодоления порога 90%+ val accuracy необходим переход на Transfer Learning (MobileNetV2 / ResNet18) с предобученными ImageNet-весами.
